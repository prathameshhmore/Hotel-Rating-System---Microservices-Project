Project Title: Hotel Rating System (Microservices Project)

Project Description: This a microservices architecture project using Spring Boot with services for User, Hotel, and Rating. 
Implemented Resilience4j patterns (circuit breaker, retry, rate limiter) for fault tolerance and reliability. 
Integrated Spring Cloud Config Server for centralized configuration and API Gateway for smart routing. 
Enabled service discovery with Eureka and improved inter-service communication using RestTemplate, resulting in 
a scalable, resilient, and highly observable system.
